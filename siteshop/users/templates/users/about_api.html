Про API

На сайте реализована возможность обращения к правилам доступа по API.

Такая возможность доступна только администратору.

Доступны следующие endpoints:

  • GET запрос (/users/api/v1/rules/) - Получение списка правил
  • GET запрос (/users/api/v1/rules/<pk>) - Получение конкретного правила по pk
  • POST запрос (/users/api/v1/rules/<pk>) - Изменение конкретного правила по pk
